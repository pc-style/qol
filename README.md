# Markdown Converter Userscript

Select DOM elements, navigate with arrows, convert to Markdown (Turndown+GFM), Readability mode, floating toolbar + settings.

## Installation

Install via Tampermonkey/Greasemonkey and it will auto-update from this repo.

## Features

- DOM element selection with visual overlay
- Arrow key navigation
- Convert selection to Markdown
- Readability mode for article extraction
- Floating toolbar
- Customizable settings
